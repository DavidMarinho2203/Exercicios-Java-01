# 🏦 Simulador de Caixa Eletrônico

Aplicação de terminal feita em Java para simular um sistema bancário simples.

## 📄️ Lista de funcionalidades:

* Criar conta
* Fazer depósito
* Realizar saque
* Consultar saldo
* Transferência entre contas
* Ver extrato de transações
